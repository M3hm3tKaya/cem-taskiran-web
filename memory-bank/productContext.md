# Product Context — HLE.IO Clone

## Neden
Ödüllü bir web deneyimini teknik olarak yeniden üretmek, ileri düzey web teknolojilerini (Three.js, GSAP, WebGL) bir arada kullanmak.

## Problem
Kompleks animasyon akışlarını (intro → 3D sahne → ana içerik) kusursuz geçişlerle birleştirmek.

## UX Hedefleri
- Sinematik, retro-fütüristik, monokrom estetik
- Kullanıcı bir "film deneyimi" yaşamalı
- Intro → TV → Main akışı kesintisiz

## Tasarım İlkeleri
- Koyu tema (#1A1A1D / #FFFFFF)
- Monospace tipografi (IBM Plex Mono)
- Grid desenleri
- Retro TV estetiği
